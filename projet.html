BusJam :

Un puzzle de logique où tu dois libérer des bus coincés dans une grille, les garer dans un parking, puis charger les bons passagers selon leur couleur.

Comment ça marche ? - Les 4 étapes :

  1. Tu cliques sur un bus — s'il a le chemin libre dans sa direction (U, D, L ou R), il quitte la grille et rejoint le parking.
  2. Le bus attend dans le parking — il reste garé jusqu'à ce que des passagers de sa couleur montent dedans.
  3. Les passagers montent un par un — le premier de la file marche automatiquement vers le bus de sa couleur dans le parking.
  4. Bus plein = bus qui repart — quand un bus atteint sa capacité max, il libère sa place dans le parking.

    5. L'objectif : vider la grille ET la file.

Les obstacles et situations de blocage :

  • Bus bloqué: un autre bus se trouve dans la trajectoire.
  • Parcking plein.
  • Mauvaise couleur le premier passeger de la file d'attente n'as pas de de bus de sa couleur dans le parking.

Condition de VICTOIRE :

  • Vider le parcking de tous les bus en les coordonnant avec les bons passagers qui attendent sur le quai.

Partie PERDUE :

  • Si le parcking est plein + prochain passager est sans bus disponible.
